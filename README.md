# 🖐️ Ishara: Real-Time Egyptian Sign Language Translation System

**Ishara** is Egypt’s **first continuous Egyptian Sign Language translation system**, designed to bridge communication gaps for the **deaf and hard-of-hearing community**.  
The system translates sign language gestures to text — and vice versa — in **real time**, providing an essential step toward accessibility and inclusion.  

---

## 🎯 Project Overview

Ishara combines the power of **Machine Learning (ML)**, **Computer Vision (CV)**, and **Natural Language Processing (NLP)** to deliver a **seamless, continuous translation experience**.  
Our mission is to make everyday communication easier and more inclusive for sign language users and non-signers alike.  

---

## ✨ Key Features

- ⚡ **Real-Time Translation:** Instantly converts Egyptian Sign Language gestures into text — and text into sign language.  
- 🧠 **Deep Learning Models:** Custom-trained models for recognizing continuous and dynamic gestures.  
- 👁️ **Computer Vision & NLP Integration:** Detects hand movements and facial expressions to improve accuracy.  
- 💬 **User-Friendly Interface:** Simple, intuitive, and accessible for all users.  
- 🤝 **Community Collaboration:** Built with feedback from sign language experts to ensure cultural and linguistic precision.

## 🎥 Data Collection

To achieve high translation accuracy, the Ishara team built a **comprehensive dataset of Egyptian Sign Language videos**.  
This dataset was used to **train, test, and validate** the machine learning models.  

**Workflow:**  
1. Capture continuous sign language gestures through a video feed.  
2. Process each frame using trained ML models.  
3. Display real-time text translations in the interface.  

---

## 💡 How to Use

1. Run the system and allow access to your camera.  
2. Use the video feed to perform continuous sign language gestures.  
3. View the real-time translation in the text box or output panel.
